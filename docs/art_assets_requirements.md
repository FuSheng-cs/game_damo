# DAMO 美术资源需求清单 (Art Assets Requirements)

为了配合“潜意识模拟器”的世界观与全新的多重结局体系，以下是本游戏所需的完整美术资源清单。所有资源建议采用**像素风 (Pixel Art)** 或 **带有明显噪点/VHS滤镜的赛博写实风**，以契合“数据幽灵”与“潜意识”的主题。

## 1. 核心人物与场景图 (Character & Scene CGs)

由于本作的页面设计已经优化为**“场景+人物一体化”**的动态背景全屏铺满模式，因此不再需要单独的纯背景图和抠底立绘，而是需要**完整的带背景场景图**。

## 4. 美术风格总结建议
*   **色彩心理学**：主色调使用**深灰/黑**与**霓虹紫/蓝**的对比。随着真相的揭露，可以逐渐加入**刺眼的红色**（代表车祸/血/崩溃）。真结局使用**纯白/晨光色**。
*   **构图**：强调玩家（男主）的主观视角（第一人称），让玩家感觉到是“自己”站在天台上看着她。
