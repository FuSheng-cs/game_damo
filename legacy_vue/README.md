# DAMO：天台上的十句话

> “在夜色中记录人生百态，却唯独找不到自己存在的意义。”

## 游戏简介

《DAMO》是一款由 AI 驱动的沉浸式心理叙事文字冒险游戏。

在这个霓虹闪烁的深夜，你来到了天台。坐在围栏边缘的是“艾”——一个紫色内染发、指间夹着忽明忽暗香烟的女孩。她是一个极度虚无的独立摄影师，带着一种颓废的破碎感，仿佛随时都会从这里滑落。

你第一眼就被她深深迷住。你想要了解她，拯救她。

**你的目标很简单，也很残酷：**
你初始只有 **10 句话** 的机会与她交谈。
你要在这个倒计时结束前，通过自由的文本对话，试着走进她的内心，解开她的心结。

## 游戏特色

*   **真正的“自由对话”**：告别传统的 A/B/C 选项。在这里，你可以输入任何你想说的话。你的语气、你的共情能力、你选择的话题，都将由底层的 AI 引擎实时解析，并决定“艾”的反应。
*   **不确定的叙述者**：她不会轻易向你敞开心扉。面对你的打探，她可能会用冷漠转移话题，可能会用沉默应对，甚至会对你虚伪的“说教”感到厌烦。
*   **动态回合与好感度系统**：如果你的话语真正触动了她，或者让她觉得有趣，她对你的好感度会增加，同时你会**额外获得 1 次对话机会**。只要你足够懂她，对话就可以一直延续下去。
*   **后日谈聊天**：如果你奇迹般地拯救了她，游戏将解锁一个全新的类似微信聊天的界面，你可以在那里与她进行日常的自由对话。

## 游玩指南

1.  **倾听大于说教**：不要一上来就居高临下地告诉她“生命很美好”。试着像对待一个疲惫的朋友一样，和她聊聊晚风，聊聊远处的霓虹灯。
2.  **留意线索**：如果你不知道该说什么，可以点击左上角的 `💡 寻找线索`，获取旁白提示（每局限 3 次）。
3.  **准备 API Key**：为了获得完整的沉浸式 AI 对话体验，请在首页的“游戏设置”中，填入你的 [阿里云千问 API Key](https://help.aliyun.com/zh/dashscope/developer-reference/activate-dashscope-and-create-an-api-key)。如果不填写，游戏将以预设的模拟对话模式运行。

## 结局

游戏包含三个基于你对话深度的结局：死亡、消失、相识。普通的套话只会让她离你远去，而冷漠的刺痛会让她坠落。
唯有灵魂的共振，才能抵达真正的救赎。
