# 角色核心提示词优化总结

## 优化概述

本次优化针对《天台十句》游戏的角色提示词系统进行了全面改进，使其更加符合游戏性，提供更丰富的情感反馈和更明确的玩家引导。

## 主要改进点

### 1. 角色设定深化

**原有问题：** 角色设定较为笼统，缺乏具体的心理锚点供玩家切入。

**优化改进：**
- 添加了具体的摄影偏好（喜欢拍雨夜的街道、凌晨的便利店）
- 明确了角色心理细节（口袋里有孤儿院的旧照片）
- 设定了角色讨厌的话语（"一切都会好起来的"）
- 确立了角色核心需求（需要被"看见"，而不是被"拯救"）

### 2. 对话阶段设计

**新增功能：** 根据好感度自动划分对话阶段：
- **疏离期（0-4分）**：冷淡、简短、保持距离
- **试探期（5-14分）**：开始观察，偶尔反问
- **松动期（15-24分）**：主动分享，语气柔软
- **信任期（25+分）**：敞开心扉，展现脆弱

### 3. 明确的好感度触发标准

**改进前：** 标准模糊，仅说明"真正触动内心"。

**改进后：**
- ✅ 明确的好感触发条件（共情、兴趣、陪伴、洞见、有趣）
- ❌ 绝对不触发的雷区（说教、轻浮、无营养、怜悯）

### 4. 情绪波动标签

**新增功能：** 4种情绪标签用于增强角色表现力：
- [情绪:刺痛] - 玩家的话戳中了痛处
- [情绪:惊讶] - 玩家的话让她意外
- [情绪:柔软] - 玩家的话让她心中一暖
- [情绪:好奇] - 玩家让她产生了兴趣

### 5. 量化结局触发条件

**改进前：** "极高的好感度"没有明确定义。

**改进后：**
- 好感度 ≥ 30 分 → [结局:相识]
- 0分 < 好感度 < 15分 → [结局:消失]
- 好感度 ≤ 0分 或 恶意挑衅 → [结局:死亡]

## 前端改动

1. **GameStore 扩展**
   - 新增 `currentEmotion` 状态用于跟踪当前情绪
   - 改进消息解析逻辑，支持情绪标签
   - 更新存档/读档逻辑以支持新状态

2. **游戏界面增强**
   - 新增好感度进度条（0-30分），直观显示与角色的心理距离
   - 添加情绪指示器，带有不同颜色和动画效果
   - 情绪标签支持渐变和脉冲动画，增强视觉反馈

## 游戏性提升分析

### 更清晰的玩家目标
- 玩家现在有明确的心理锚点可以切入（摄影、雨天、孤独感）
- 好感度进度条给玩家提供了清晰的成长反馈
- 明确的雷区警告让玩家避免无效对话

### 更丰富的情感曲线
- 4个对话阶段提供了完整的情感递进体验
- 情绪标签让角色反应更加立体、真实
- 每个阶段的语气变化都能让玩家感受到进展

### 更好的游戏平衡
- 量化的结局条件让游戏更公平
- 30分的阈值让"相识"结局既难又可达成
- 10句基础对话 + 最多额外6句（10轮）= 平衡的对话长度

## 测试建议

1. 测试各阶段对话的语气变化是否自然
2. 验证好感度触发和扣除逻辑是否正确
3. 检查情绪标签的视觉反馈是否合适
4. 测试不同结局的触发条件是否准确
5. 确保存档/读档功能支持新状态
